git life cycle:
untracked (arquivos não adicionados), unmodified(arquivos inalterados), modified (arquivos modificados), stage(arquivos add)

CONFIGURAÇÕES BÁSICAS DE USUÁRIO:

git config --global user.name "Cesar"
git config --global user.email "paulo@mail.com"
git config --global core.editor subl

INICIALIZAÇÃO DO GIT:
git init

VERIFICAÇÃO DO PROJETO:
git status
git log

ADD ARQUIVOS AO GIT: (passa o file para "stage")
git add <file>

CRIA UMA VERSÃO DO PROJETO:
git commit -m "Algum comentário para o commit"
Output:
-> [master (root-commit) ad46a3f] Add readme.md
   1 file changed, 3 insertions(+)
   create mode 100644 readme.md
ad46a3f é o hash que identifíca a versão.


PEGAR A CHAVE SSH:
cd ~/.ssh/
cat id_rsa.pub

CONECTA O REPOSITÓRIO LOCAL COM O REPOSITÓRIO REMOTO
git remote add origin https://github.com/CesarCode/git-course.git
git push -u origin master
